# Day 62 — Inventory Management System

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A full stack inventory management system with stock
tracking, low stock alerts and transaction history.

## How to run
```bash
pip3 install flask
python3 app.py
```
Then open http://localhost:5000

## Features
- Add and manage products with categories
- Stock in/out transactions
- Low stock alerts
- Search and filter products
- Transaction history log
- SQLite database
